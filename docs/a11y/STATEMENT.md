# Accessibility statement

Outcome Receipts targets WCAG 2.2 AA for the generated trace view and accessible
chart output. Automated checks cover axe, pa11y, Lighthouse, reflow, and reduced
motion. The trace is a self-contained offline page with semantic tables, headings,
receipt details, and locale metadata.

Known gap: the current release evidence does not yet include completed NVDA and
VoiceOver walkthroughs. See the dated walkthrough and ACR in this directory.

Report an accessibility barrier through a GitHub issue without attaching real
service data, client identifiers, or report exports. Security or privacy issues
must use the private channel in `SECURITY.md`.
